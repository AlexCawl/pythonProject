Контрольные вопросы

1. Напишите, какое значение будет иметь переменная Z после выполнения операторов:
    Z=0
    if X>0:
        if Y>0 :
            Z=1
        else:
            Z=2
    при следующих значениях переменных X и Y:
    a) X=1, Y=1           Z = 1
    b) X=1, Y=-1          Z = 2
    c) X=-1, Y=1          Z = 0

2. Пусть имеется оператор присваивания X=A or B and C, где переменные X, A, B, C имеют тип Boolean.
   Напишите условный оператор, который эквивалентен данному
   (например, оператору присваивания X=not A эквивалентен условный оператор if A : X=False else X=True).
    if A == False:
        if B == False:
            if C == False:
                X = False
            else:
                X = True
   Пусть имеется условный оператор if A : X=B else X=C, где переменные X, A, B, C имеют тип Boolean.
   Напишите оператор присваивания, который эквивалентен данному.
    X = A and B or not A and C
   Напишите синтаксис условного оператора.
    if A:
        X = a
    elif B:
        X = b
    else:
        X = c

    Вычислите выражения:

        123 // 10 = 12

        -123 // 10 = -13

        123 % -10 = -7

        -123 % -10 = -3

Объясните результат. Какой результат выдается интрепретатором Python?
Знак остатка совпадает со знаком делителя. Вычисляется по математическим правилам.
