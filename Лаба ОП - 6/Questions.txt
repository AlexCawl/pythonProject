Контрольные вопросы

    Напишите синтаксис цикла for? Опишите семантику оператора.
        for <name> in [a, b, c, d, e, f, g, ..., z]:
            <program body>

        Значения последовательности (списка) являются значениями параметра цикла, соответственно параметр цикла будет принимать все значения последовательности (списка);
        Число шагов цикла равно количеству значений, принимаемых его параметром, то есть длине последовательности (списка).

    Напишите синтаксис цикла while? Опишите семантику оператора.
        while <statement>:
            <program body>

        Вычисляется значение условия и если оно не равно False, то выполняется тело цикла;
        В противном случае выполнение оператора цикла завершается.
            
    Каким образом модуль подключается к программе? 
        import <module_name>
        import <function> from <module_name> as <function_new_name>

    Поясните значение строки программы: if __name__ == "__main__":
        Когда интерпретатор Python читает исходный файл, он исполняет весь найденный в нем код. Перед тем, как начать выполнять команды, он определяет несколько специальных переменных. 
        Например, если интерпретатор запускает некоторый модуль (исходный файл) как основную программу, он присваивает специальной переменной __name__ значение "__main__". 
        Если этот файл импортируется из другого модуля, переменной __name__ будет присвоено имя этого модуля.

    Что такое __name__, __main__?
        Специальная системная переменная, и системное значение соответственно.

    Каким образом можно открыть файл на чтение? на запись?
        file = open('<file_name>', 'r')
        file = open('<file_name>', 'w')

    Что произойдет, если мы с помощью функции open попробуем открыть несуществующий файл?
        Если на чтение: FileNotFoundError: [Errno 2] No such file or directory: '<file_name>'
        Если на запись: Программа создаст в своем каталоге файл с указанным именем.

            s=s+1/i

        s=1

           s=s+1/i          # s = 0;          в цикл даже не заходит
